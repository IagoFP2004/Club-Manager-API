INSTRUCCIONES PARA CONFIGURAR EL ENVIO DE CORREOS

PASO 1: Crear archivo .env
===========================

Crea un archivo llamado ".env" en la raíz de tu proyecto con el siguiente contenido:

# Database Configuration
DATABASE_URL="postgresql://app:!ChangeMe!@127.0.0.1:5432/app?serverVersion=16&charset=utf8"

# Mailer Configuration - Gmail SMTP (sin verificación de dominio)
MAILER_DSN=gmail://TU-EMAIL@gmail.com:TU-APP-PASSWORD@default
MAILER_FROM_EMAIL=TU-EMAIL@gmail.com

# Symfony Configuration
APP_ENV=dev
APP_SECRET=your-secret-key-here

PASO 2: Configurar Gmail
========================

1. Ve a https://myaccount.google.com/
2. Seguridad > Verificación en 2 pasos (debe estar activada)
3. Contraseñas de aplicaciones > "Generar contraseña de aplicación"
4. Selecciona "Correo" y "Otro (nombre personalizado)"
5. Escribe "Symfony App" como nombre
6. Copia la contraseña generada (16 caracteres como: abcd-efgh-ijkl-mnop)

PASO 3: Actualizar .env
=======================

Reemplaza en tu archivo .env:
- TU-EMAIL@gmail.com → tu email real de Gmail
- TU-APP-PASSWORD → la contraseña de aplicación generada

Ejemplo real:
MAILER_DSN=gmail://miguel@gmail.com:abcd-efgh-ijkl-mnop@default
MAILER_FROM_EMAIL=miguel@gmail.com

PASO 4: Reiniciar el servidor
=============================

Después de crear el archivo .env, reinicia tu servidor Symfony:
php bin/console cache:clear

¡LISTO! Ahora podrás enviar correos sin verificar dominio.
